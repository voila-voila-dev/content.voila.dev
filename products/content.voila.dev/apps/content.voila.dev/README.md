# @voila/content.voila.dev

Marketing site for **content.voila.dev** — the open-source, TanStack-native
headless CMS. Built with TanStack Start, deployed on Cloudflare Workers.

## Local development

From the repo root (the Bun workspace owns the install):

```bash
bun install
bun --filter @voila/content.voila.dev dev
```

The site runs at <http://localhost:3000>.

## Build

```bash
bun --filter @voila/content.voila.dev build
```

Output: `dist/`, plus a Cloudflare Worker entry generated by
`@tanstack/react-start`'s Cloudflare adapter.

## Deploy

```bash
bun --filter @voila/content.voila.dev deploy
```

Requires `CLOUDFLARE_API_TOKEN` and `CLOUDFLARE_ACCOUNT_ID` in the environment.
Worker name and bindings live in [`wrangler.jsonc`](./wrangler.jsonc).

## Structure

```
src/
├── routes/
│   ├── __root.tsx          # SEO meta, dark theme shell, structured data
│   └── index.tsx           # landing page (composes the sections)
├── site/
│   ├── config.ts           # site-wide constants (links, commands)
│   ├── logo.tsx            # gradient wordmark + monogram
│   ├── site-header.tsx     # sticky nav + mobile Sheet menu
│   ├── site-footer.tsx     # 4-column footer
│   ├── section.tsx         # Section / SectionHeading / SectionLead primitives
│   ├── copy-command.tsx    # one-line copy-to-clipboard for the bootstrap cmd
│   └── sections/           # 11 page sections
└── styles.css              # imports @voila/ui/tailwind, dark-mode overrides
```

UI primitives come from [`@voila/ui`](../../../../packages/ui). The dark theme
overrides a handful of CSS variables in `styles.css`; the rest is shared with
the admin design system.
